## Architecture

- `Sources/NovotroOpera`
  - unified shell, top tabs, shared project session, shared loading overlay
- `Sources/NovotroWrite`
  - Write workspace and libretto staging scratchpad
- `Packages/NovotroProjectKit`
  - local project loading, loading progress, shared Opera chrome
- `Packages/NovotroScore`
  - Score workspace vendored locally for the unified app
- `Packages/NovotroAnimate`
  - Animate workspace vendored locally for the unified app
