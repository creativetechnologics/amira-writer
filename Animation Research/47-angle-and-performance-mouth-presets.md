# 47 — Angle and Performance Mouth Presets

Date: 2026-03-31

## Purpose
Extend mouth presets beyond character identity into angle family and performance mode.

## Axes
Each mouth preset should be influenced by:
- character
- angle family
- mode: speech or singing
- emotional preset

## Angle family differences
### Front
- highest visual clarity
- widest usable shape range
- best for expressive vowels

### Quarter-left / Quarter-right
- moderate compression
- preserve clarity but reduce overly horizontal widening
- good default production angle

### Profile-left / Profile-right
- simplify heavily
- prioritize silhouette readability
- limit tongue/teeth detail

## Performance mode differences
### Speech
- quicker closures
- lighter sustain
- more consonant visibility

### Singing
- stronger vowel holds
- softer phrase releases
- emotional modifier has more visual influence

## Recommended combination model
Final mouth behavior should be:
base viseme family
+ angle modifier
+ performance modifier
+ emotion modifier

That keeps the system modular and avoids exploding the total asset count.
