# Prototypes

Research-only schema scaffolding for the future animation system.

## Files
- `motion_plan_schema.json` — sparse motion-plan contract for LLM-authored shots
- `character_package_vnext_schema.json` — richer package schema with costume packs, mouth profiles, primitives, and QA metadata
- `mouth_profile_schema.json` — separate mouth-engine contract

These files are not wired into the app yet.
