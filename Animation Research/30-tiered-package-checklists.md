# 30 — Tiered Package Checklists

Date: 2026-03-31

## Purpose
Provide concrete per-tier checklists so package scope does not drift blindly.

## Hero checklist
### Must have
- approved master sheet
- approved head sheet
- approved body sheet for default costume
- mouth profiles for at least front + quarter-left + quarter-right
- 10+ body primitives
- 1 accessory/detail sheet if props matter
- approved QA and promotion history

### Should have
- second costume pack
- profile-left and profile-right mouth profiles
- additional emotional gestures
- corrective assets for hero closeups

### Nice to have
- specialty overlays
- alternate expression families
- rare shot-specific correctives

## Supporting checklist
### Must have
- approved master sheet
- approved head sheet
- approved default costume body sheet
- front mouth profile
- 5+ body primitives

### Should have
- quarter-turn mouth profiles
- one additional costume if the script needs it
- moderate expression coverage

## Background checklist
### Must have
- approved identity/body reference
- one costume sheet
- minimal motion primitives

### Should have
- only the mouth coverage actually needed by the shot list

## Production rule
Do not build hero-level packages for every character by default.
Promote a character to a higher tier only when the scene load justifies it.
