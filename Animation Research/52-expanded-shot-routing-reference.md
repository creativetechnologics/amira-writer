# 52 — Expanded Shot Routing Reference

Date: 2026-03-31

## Purpose
Collect a wider range of concrete routing examples so later implementation can start from tested defaults.

## Internal-biased cases
- dialogue close-up
- singing medium close-up with approved mouth package
- solo reaction shot
- restrained walk-and-talk

## Hybrid-biased cases
- duet medium shot
- ensemble dialogue where only leads need precise acting
- group walk with featured leads
- FX transition anchored by internal character acting
- action insert where the lead remains readable but environment complexity rises

## AI-video-fallback-biased cases
- crowd spectacle wide
- chase wide with moving camera
- atmospheric montage with heavy environmental dynamics
- destruction-heavy or battle-heavy wides

## Rule of thumb
If precise retakes and continuity dominate, bias toward internal.
If spectacle and camera/environment complexity dominate, bias toward hybrid or fallback.
