# 99 — Ensemble Beat Lighting Diff Bundle

Date: 2026-03-31

## Purpose
Define the review bundle for revised ensemble beat-lighting plans.

## Bundle contents
An ensemble beat-lighting diff bundle should contain:
- baseline ensemble beat-lighting fixture JSON
- revised ensemble beat-lighting fixture JSON
- `ensemble_beat_lighting_diff_report.json`
- `ensemble_beat_lighting_diff_report.md`

## First pilot requirement
The handoff packet should include:
- one passing ensemble beat-lighting diff example
- one regressed ensemble beat-lighting diff example
- readiness reports for the same baseline/revision pair

## Key reporting rule
The diff bundle should surface both:
- structural regressions in the plan itself
- routing-pressure changes implied by the revised plan
