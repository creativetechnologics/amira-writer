# 37 — Character Mouth Emotion Presets

Date: 2026-03-31

## Purpose
Define per-character emotional mouth tendencies so the mouth engine can feel character-specific instead of generic.

## Luke Hart
### Default speech bias
- restrained
- practical
- moderate tension under stress

### Singing bias
- opens wider on resolute/heroic lines
- limited chatter
- concern and fatigue should soften release more than widen corners

## Amira Nazari
### Default speech bias
- softer release
- elegant rounded transitions
- stronger emotional nuance in corner tension

### Singing bias
- sustained vowels should feel expressive and lyrical
- smile and sorrow modifiers matter more than on Luke
- avoid harsh clipped closures unless the moment is intentionally severe

## Yasmin Nazari
### Default speech bias
- lighter, smaller openings
- more restrained than Amira
- softer concern/sadness modifier set

### Singing bias
- if elevated to singing coverage, begin with a simplified lyrical profile before building a full hero-grade set

## Recommendation
Store these as lightweight preset tables that modify:
- openness
- release softness
- smile tendency
- strain allowance
- asymmetry tolerance
