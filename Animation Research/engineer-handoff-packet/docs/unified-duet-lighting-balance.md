# 74 — Unified Duet Lighting Balance Rules

Date: 2026-03-31

## Purpose
Describe how a single location light should balance Luke and Amira when they appear together.

## Shared-light principle
Luke and Amira should feel like they occupy the same location, same time of day, and same practical-light system.

## Difference-preservation principle
Even under one shared light world, the runtime should preserve:
- Luke's utility-fabric and shoulder/jaw silhouette readability
- Amira's face-framing, scarf contour, and softer facial readability

## Duet balancing rules
1. choose one shared profile first
2. preserve the shared directionality of light
3. use only limited character-local protections
4. never let local protections contradict the location light source
5. prioritize face and mouth readability in dialogue/singing moments

## Example
In `moonlight_blue` village-night scenes:
- Luke may accept stronger cool shadow shaping
- Amira may require more fill protection around eyes, mouth, and scarf edge

That is acceptable as long as the light still reads as coming from the same night world.
