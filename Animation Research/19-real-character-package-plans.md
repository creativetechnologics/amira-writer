# 19 — Real Character Package Plans: Luke, Amira, Yasmin

Date: 2026-03-31

## Purpose
Pressure-test the research model against real Amira characters.

## Assumptions
- packages are intended for semi-autonomous 2D animation
- difficult hero shots may still route to AI video fallback
- counts below are planning targets, not exact locked requirements
- package totals include master sheets, head/body sheets, gesture banks, mouth assets, accessories, and a modest corrective reserve

## Luke Hart
### Tier
Hero

### Likely costume packs
- medic desert / military field
- plain clothes / civilian
- optional clinic/apron or damaged-wardrobe variant later

### Planning count
- base package: **110–140 assets**
- each extra costume: **20–28 assets**

### Why high
Luke likely needs:
- full dialogue coverage
- singing coverage
- walk-and-talk coverage
- hand/prop interaction with satchel and medical gear
- emotional close-up coverage

### Cost at Nano Banana 2
At 2K:
- standard base: about **$11.11–$14.14**
- batch base: about **$5.50–$7.00**
- extra costume batch: about **$1.00–$1.40**

## Amira Nazari
### Tier
Hero

### Likely costume packs
- civilian default
- ceremony / formal
- optional travel or hardship variant later

### Planning count
- base package: **105–135 assets**
- each extra costume: **18–26 assets**

### Why high
Amira is likely to require:
- expressive singing coverage
- emotional closeups
- culturally and silhouette-sensitive costume handling
- scarf / hair / garment overlays that must stay elegant

### Cost at Nano Banana 2
At 2K:
- standard base: about **$10.61–$13.64**
- batch base: about **$5.25–$6.75**
- extra costume batch: about **$0.90–$1.30**

## Yasmin Nazari
### Tier
Supporting, with potential hero-like scenes if upgraded later

### Likely costume packs
- civilian default
- optional ceremony / event variant

### Planning count
- base package: **55–80 assets**
- each extra costume: **10–16 assets**

### Why moderate
Yasmin may need:
- dialogue coverage
- moderate emotional range
- lighter locomotion and fewer hero-specific correctives than Luke or Amira

### Cost at Nano Banana 2
At 2K:
- standard base: about **$5.56–$8.08**
- batch base: about **$2.75–$4.00**
- extra costume batch: about **$0.50–$0.80**

## Best first production order
1. Luke base + one costume
2. Amira base + one costume
3. Yasmin base + one costume
4. second costumes for Luke and Amira
5. expand accessory and corrective banks only after real scene testing

## Planning recommendation
Budget generation by milestone, not by whole-film total:
- identity lock
- first costume package
- mouth package
- gesture expansion
- second costume

That reduces waste and lets QA/review guide what is truly needed.
