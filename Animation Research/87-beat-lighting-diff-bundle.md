# 87 — Beat Lighting Diff Bundle

Date: 2026-03-31

## Purpose
Define the review bundle for beat-level lighting plans.

## Bundle contents
A beat-lighting diff bundle should contain:
- baseline beat-lighting plan JSON
- revised beat-lighting plan JSON
- `beat_lighting_diff_report.json`
- `beat_lighting_diff_report.md`

## First pilot requirement
The handoff packet should include:
- one passing beat-lighting diff example
- one regressed beat-lighting diff example
- one single-plan continuity audit output
