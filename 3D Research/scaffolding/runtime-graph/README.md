# Runtime Graph Scaffold

Purpose: describe the minimum Amira-native 3D runtime state.

## Runtime graphs

### WorldGraph
- active world
- loaded environment assets
- placement transforms
- world toggles

### ShotGraph
- active shot
- cameras
- timing
- focus state

### PerformanceGraph
- characters
- body tracks
- face tracks
- mouth tracks

### StyleGraph
- cel shader profile
- outlines
- lighting rig
- atmosphere
- color grade

### ReviewGraph
- pending commands
- validation warnings
- preview diff
